<screen> Directive

Reduce the redundancy html template, according to the program-id and theme setting in config file. this example will download the pageview template from Templates/screen/dw01dp.html

Usage
- Use inside of entry
- Use inside of pageview 

			<pageview ng-model="deptDataList" program-id="dw01dp">
				<screen program-id="dw01dp"></screen>
			</pageview>
ng-model = object
prorgam-id = string
