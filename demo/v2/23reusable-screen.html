Reusble Screen

Case Details:
Subject:
Directive screen - customize the screen with same function on different pages
Business Logic:
  • Using the parent program-id to define the screen-id
  • Override the parent program-id by the screen-id
Procedures:
  1. Default load the screen via parent program-id
  2. Override the program-id by screen-id
  3. Auto shows data on page loaded
  4. Click the navigation buttons to navigate to pages

Default use parent program-id dw01dp as the screen template

Override the parent program-id, use screen-id dw02dp as the screen template
