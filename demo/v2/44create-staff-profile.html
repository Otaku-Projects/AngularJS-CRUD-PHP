Create Staff Profile

Case Details:
Subject:
Directive entry, editbox - list, create staff record
Business Logic:
  • Editbox auto loads the department records
  • Select to the department record on pageview
  • The selected department record will shows
Procedures:
  1. Input the Staff ID
  2. Input the Surname (optional)
  3. Input the Given Name (optional)
  4. Input the Chinese Name (optional)
  5. Select the Department (optional)
  6. Select the Date of Birth (optional)
  7. Select the Employment Date (optional)
  8. Click the [Create] button
