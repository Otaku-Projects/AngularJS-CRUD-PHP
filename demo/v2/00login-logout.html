Redirect if has not login

RequiresAuthorization

Send http request to the server, check if the webpage vistor doesn't logged in redirect to the login page.
You may change the redirect destination, please see how to config.

Usage
Put this code to the page which provide features for member only 

                app.run(function ($rootScope, Security) {
                    Security.RequiresAuthorization();
                });

Redirect if logged in

GoToMenuIfSessionExists

Send http request to the server, check if the webpage vistor has logged in, redirect to the main page of the member.
You may change the redirect destination, please see how to config.
Example

Put this code to the page which provide features for member only 

                app.run(function ($rootScope, Security) {
                    Security.GoToMenuIfSessionExists();
                });
