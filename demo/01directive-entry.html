<entry> Directive

Handle the crud action for entry form, create the binding between the form control and the form data object.

Usage

- As a data form for input record.

Arguments

Param Type Details
ng-model expression assignable expression to bind to
program-id string Name of the program id,
edit-mode string Specify the mode [create | view | amend | delete] of entry, different mode has different behaviors.
