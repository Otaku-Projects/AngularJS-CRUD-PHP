<entry> Directive

Handle the crud action for entry form, create the binding between the form control and the form data object.

Usage
As a data form for input record. 

                <entry ng-model="entryForm" program-id="hs02dp" edit-mode="create">
                  <form ng-submit="SubmitData()">
                    <fieldset>
                      <div class="form-group">
                        <label for="inputDeptCode">Department Code</label>
                        <input type="text" class="form-control" id="inputDeptCode" ng-model="entryForm.DepartmentCode">
                      </div>
                      <div class="form-group">
                        <label for="inputDesc">Description</label>
                        <textarea class="form-control" id="inputDesc" ng-model="entryForm.Desc"></textarea>
                      </div>
                      <div class="form-group">
                        <button type="submit" class="btn btn-default">Create</button>
                      </div>
                    </fieldset>
                  </form>
                </entry>
ng-model = object
prorgam-id = string
edit-mode = [create | view | amend | delete]
