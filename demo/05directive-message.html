<message> Directive

Display the message for the entry form, inquire result and process result on the page. By default, the CRUD action result will auto put into the message directive. If you put the message directive on the top and the page too long, you may required to call javascript scroll to the top action after each CRUD response

Usage
- Use inside of entry
- Use inside of pageview 
<message ng-model="processMsg" auto-close>
  <div class="well well-sm" ng-if="msgCtrl.ngModel.length > 0">
	<div ng-repeat="dspMsg in msgCtrl.ngModel track by $index" ng-bind="dspMsg"></div>
  </div>
</message>
ng-model = object
prorgam-id = string
OpenPageView() = method
