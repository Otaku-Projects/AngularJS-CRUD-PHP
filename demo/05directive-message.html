<message> Directive

The result of entry action will auto bring the message to this directive, the message will auto floating on the page. You may use this directive to display the in-line message.

If you put the message directive on the top and the page too long, you may required to call javascript scroll to the top action after each CRUD response

Usage
- Display the inline message

Arguments

Param Type Details
ng-model expression assignable expression to bind to
auto-close string auto close the message

Methods

Method Return Value Description
Initialize() null Initialize the directive
InitScope null Initialize the directive scope
InitDirective null Build the default behaviors
