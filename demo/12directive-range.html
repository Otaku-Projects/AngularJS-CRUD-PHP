<range> Directive

For the report / inquire / process form which required range criteria selction. Select record from the editbox.
The default value of start is All, the end control is locked.
The directive compare the string in alphabetically

Usage
- Use inside of entry

Arguments

Param Type Details
ng-model expression assignable expression to bind to
range string [start | end], actual this is the editbox argument, specify the start/end control for create a watch listener
range-value string assignable expression to bind to, actual this is the editbox argument, specify the start/end value for create range behaviour

Methods

Method Return Value Description
Initialize() null Initialize the directive
InitScope null Initialize the directive scope
InitDirective null Build the default behaviors
