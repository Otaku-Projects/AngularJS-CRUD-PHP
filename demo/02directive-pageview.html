<pageview> Directive

Display a data set in pagination view, use this element for viewing records and/or select record.

Usage
- Use this element as a view for selecting one record for amendment.
- This element may reuse in <editbox> directive, to display foreign key table records and allowed to select one in <entry> form 

				<pageview ng-model="deptDataList" program-id="dw01dp">
				  <div class="table-responsive">
				    <table class="table table-condensed table-hover">
				      <thead>
				        <tr>
				          <th>Department Code</th>
				          <th>Description</th>
				        </tr>
				      </thead>
				      <tbody>
				        <tr ng-repeat="deptRow in pageviewCtrl.ngModel" 
				        ng-click="PointedToRecord(deptRow, $event, this)"
				        ng-dblclick="SelectedToRecord(deptRow, $event, this)">
				          <td>{{deptRow.DepartmentCode}}</td>
				          <td>{{deptRow.Description}}</td>
				        </tr>
				      </tbody>
				    </table>
				  </div>
				</pageview>
ng-model = object
prorgam-id = string
