<pageview> Directive

Display a data set in pagination view, use this element for viewing records and/or select record.

Usage

- Use this element as a view for selecting one record for amendment.
- This element may reuse in <editbox> directive, to display foreign key table records and allowed to select one in <entry> form

Arguments

Param Type Details
ng-model expression assignable expression to bind to
program-id string Name of the program id
