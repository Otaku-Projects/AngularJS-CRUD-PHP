<editbox> Directive

For the entry form which required foreign key/reference key from others tables. Click the button to to view the pageview for selecting a record. this example will download the pageview template from Templates/screen/hw92nl.html

Usage
- Use in entry, specify for create form or update form

Arguments

Param Type Details
ng-model expression assignable expression to bind to
program-id string Default as the parent program id, but you may specify a new program id
screen-id string Default as the parent program id, but you may specify a new program id

Methods

Method Return Value Description
Initialize() null Initialize the directive
InitScope null Initialize the directive scope
EventListener($scope, $element, $attrs, $ctrl) null Allows to attach the javascript event.
$scope - directive scope
$element - directive element
$attrs - directive attrs
$ctrl - directive controller
This function use to attach the event or create the instance for third-party resource.
InitDirective null Build the default behaviors
PopupPageview null Display the hidden pageview
OpenPageView null Popup a modal and call PopupPageview
ClosePageView null Hide the modal and pageview
SetEditboxNgModel null Used by Pageview, bring the record to the editbox when selecting record on pageview
ClearEditboxNgModel null Used by entry, clear the editbox ngModel when form data created
